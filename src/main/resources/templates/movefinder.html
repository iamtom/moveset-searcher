Moveset Searcher

How to use: select up to four moves. Click 'submit' to see which Pokémon can learn all your chosen moves.

The tool will show which versions of the game these moves can be learned in.

Note that sometimes Pokémon can only learn all the moves in certain versions of the games.

Results that say "Level 0" indicate the move is learned when it evolves.
